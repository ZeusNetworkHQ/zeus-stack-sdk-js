# zpl-sdk-js

A modern TypeScript SDK for Node.js and browser environments, built with rolldown.

## Features
- ESM/CJS dual output
- Full TypeScript support
- Ready for npm publish

## Usage
```sh
pnpm add zpl-sdk-js
```

```ts
import { hello } from 'zpl-sdk-js';
console.log(hello('World'));
```

## Build
```sh
pnpm build
```
